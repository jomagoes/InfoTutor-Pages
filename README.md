# InfoTutor - Server Gateway Infrastructure

Este repositorio contiene la configuración estática de enrutamiento y pasarelas de enlace para la red de servidores conectados a la plataforma **InfoTutor**.

## ⚠️ Aviso de Uso Interno

* **Propósito:** Servidor de redirección estática y resolución de rutas de infraestructura marca blanca.
* **Restricciones:** El contenido de este repositorio es generado y gestionado de forma automatizada mediante API interna. Cualquier modificación manual externa romperá la sincronización de las pasarelas activas.
* **Propiedad Intelectual:** Todos los derechos reservados. Queda estrictamente prohibida la reproducción, copia o distribución de la arquitectura de rutas aquí expuesta sin la autorización expresa de la administración de InfoTutor.
